# UI Automation Framework – Selenium

This project is a **UI end-to-end automation framework** for a sample e-commerce web app.

## Tech Stack

- Java  
- Selenium WebDriver  
- TestNG (or JUnit)  
- Maven or Gradle  
- Allure / Extent Reports  
- GitHub Actions or Jenkins  

## Project Structure

```text
src/
 └─ test/
     ├─ java/
     │   ├─ pages/          # Page Objects
     │   ├─ tests/          # Test classes
     │   └─ utils/          # Helpers, drivers, config
     └─ resources/
         ├─ testdata/       # CSV / JSON test data
         └─ config/         # Environment configs
